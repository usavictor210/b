August 18th, 2020
Long ago, dan-simon made a modification of Antimatter Dimensions named Eternity Respecced.

It has been many months, 2 years since then. In November 2018, dan-simon proceeded to delete the repository of /b...
...not before the repository was passed onto me.
And so, it began.
After a long time, the Intergalactic update is finally in development, with much needed balance changes.

INTERGALACTIC: It's finally here
- Galactic studies and milestones with even more automation than before
- ...yes, there are more Dimensions too. Galactic Dimensions.
- There is also some obvious intergalactic statistics, I won't go into detail too much since we all know them at this point
- 8 MORE ETERNITY CHALLENGES (you heard that right, EIGHT of them).
After their 5th completion, Eternity Challenges can be attempted again in Time Trial mode to complete the challenge as fast as possible.
- Still no unfair scaling (we promise)
- TWO MORE ACHIEVEMENT ROWS, and a lot more new achievement images
- Many rewords and code changes to make the game more understandable
- Bulk buy is now capped at 1e10x, because who really needs 1e100x bulk buy now?
- Max all Infinity Dimension button has been added for a needed QoL change.
- The bulk buy Dimension Boost upgrade now lets you buy max Dimension Boosts possible manually
- Fixed the IP peak/min. In fact, all peak statistics have been fixed.
- Removed and added some news tickers
- Changed some text font displays
- Time is relative now has a reward giving a multiplier to Normal Dimensions based on time spent in this Eternity (effect is normally capped at 1e33x, but gets stronger as you get more Eternities. Hardcap is 1e100x).
- Made the statistics screen and all subtabs neater.
